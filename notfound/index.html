【授翻】file not found/文件已删除

2014-06-24 神探夏洛克 Johnlock无差, 华福, 福华, 翻译, 虐文, 角色死亡 0 评论 字数统计: 1.9k(字)

作者：nejem

题目是404 Error: File Not Found，然而404 error似乎没法放在题目里，不然就发不出来……

（老年痴呆AU，角色死亡暗示，分级：G）

原文：https://sherlockbbc-fic.livejournal.com/12826.html?thread=68782874#t68782874

正文：

Sherlock醒来，在床头柜上发现一张黄色的便利贴。

黑咖啡，两块糖。不加糖的茶给John，茶里面加几乎尝不出的一点牛奶。上面这么写着。

他起来，穿上他蓝色的旧晨袍，不系带子，任它在他下楼走进厨房时在他身体周围窸窣作响。他照着那张小纸上的指示泡了咖啡，黑咖啡，两块糖，然后小心地啜了一口。他喜欢这味道。

John还在睡觉，他得记着一会要给他泡茶。他拿来笔和纸，写下来，把便条留在了厨房流理台上。

茶，不加糖，一点奶。

当John叫他的时候，他在外面，在他的蜂房之间。不到一分钟之后，他听见身后草坪上有细微的脚步声，一只胳臂环在他细瘦的腰间。

“你已经在外面待了好久了。进来吧，天有点凉了，晚餐我做汤。”

你错了，Sherlock跟着John时想。John错了，不可能有那么久，十分钟之前他还在屋里泡咖啡。

在他们进厨房之前，John站在他面前，在Sherlock的额头上留下一个轻吻，试着抹平他皱起的眉头，接着便走向炉灶准备他们的晚餐。

Sherlock的目光落在流理台上，看到了另一张黄色的便利贴。茶，不加糖，一点奶。

他忘记给John泡茶了。

Sherlock醒来，在床头柜上发现一张蓝色的便利贴。

无聊无聊无聊无聊。无聊。给Lestrade打电话找个案子。无聊。

他起来，穿上他蓝色的旧晨袍，不系带子，任它在他下楼找手机时在他身体周围窸窣作响。它响了一次，两次，三次，Lestrade接了起来。Sherlock没给他任何说话的空儿。

“我感觉我的脑子就要烂掉了，我需要一个案子。”

Lestrade在电话那头叹了口气。Sherlock没法看见他，不过他一定抬起了手，并疲惫地把它按在了额头上。

“Sherlock，我退休了，记得吗？我没法给你找案子，我已经没有权限再参与调查了。”

他困惑地皱起眉毛，想，Lestrade退休了？这讲不通啊，明明就在上周他们还在一起调查连环自杀案。

他向下看，在沙发前的咖啡桌上发现了另一张蓝色的便利贴。

照照镜子。记得照照镜子。

他走向暗色的壁炉，就在那，在壁炉台上面，他看见了自己的镜像：他的头发还是卷的，向来是卷的，因为一夜安眠而乱糟糟的，成了灰色。某几处还有几绺白色。他的前额和嘴周围的线条刻上了来自忧愁、阅历、岁月和人生的皱纹。他的眼睛还闪着光，明亮而专注，它们属于活了半个世纪之久的人。

“Sherlock？你在听吗？”

他把视线从镜子移开，落回咖啡桌，落回那张蓝色的便利贴。

“你老了。”

Lestrade在电话里哼了一声，半是觉得有趣，半是疲倦。“是啊，我老了。”

“我忘了。”

“没关系，现在你记得了。”

Sherlock醒来，在床头柜上发现一张绿色的便利贴。

Mycroft死了。

他起来，穿上他蓝色的旧晨袍，不系带子，任它在他走进浴室时在他身体周围窸窣作响。John正在那里冲澡。

“Mycroft死了。”

John把水关掉，拉开浴帘，从附近拿起一条毛巾擦干自己。他从浴缸里迈出来，带着轻柔的担忧看着Sherlock，盯着他手里亮色的便利贴片刻。

“我知道。你忘了？”

Sherlock看了他好一会：他忘了。他讨厌承认这个，不过没错，他忘了。

他点头。

“谁是Mycroft？”

John看上去就像脸刚被人揍了一拳。他深吸一口气，眼睛上移盯着天花板没到一秒钟，然后回到原位看着Sherlock，他天才、疯狂、衰老、生了病的Sherlock。

“他是你的兄弟。你的哥哥，你们以前总是互相闹别扭不过谁都能看出来你们很在乎——”
他不得不住口，因为他已经哽咽了。他眨着眼，一次，两次，三次。又几次眨眼，他的眼泪回到了原处。

“把水烧上，好吗？我去穿衣服，然后如果你想，我们可以说说Mycroft。”

Sherlock看着John匆匆留给他一个小小的、忧伤的微笑。他好奇他为什么难过。他点了点头。

“好，我想我会愿意记起我的哥哥。”

Sherlock醒来，在床头柜上发现一张粉色的便利贴。

扔掉冰箱里的头。

他起来，穿上他蓝色的旧晨袍，不系带子，任它在他下楼走向冰箱时在他身体周围窸窣作响。

“早上好。”John微笑着招呼他。

Sherlock无视了他，打开冰箱，扫视着里面的一切，移动着食物和瓶子。

“你在找什么？”John问他，声音里的耐心清晰可辨。

“那颗头。”

一声叹息。Sherlock能感觉到John走到他身后来，他的手探来握住他的，制止了他徒劳的搜索。

“冰箱里已经很多年没有人头了。”

他看着John，突然想起来：过去，外面在下雨，John打开冰箱，然后因为一个人头，一只脚和一颗心脏带着厌恶退后一步。他想起John，一个更年轻的John。那个John关上冰箱，摇着头，嘟哝着单独弄一个实验用冰箱，还有要Molly永远别再让Sherlock借走别的人体部件。

Sherlock快速走进客厅找他的手机，John紧紧地跟着他，担忧地皱起眉头。

“现在你在干什么？”

“我才想起来，Molly。”

“Molly怎么了？”

“我从来没为头的事谢过她。我该给她打电话，那头很有用。

John看着Sherlock伏下身来在沙发底下找他的手机。他不想告诉他就在一周之前他们参加了Molly的葬礼。

Sherlock会想起来的，也许以后。

Sherlock醒来，在床头柜上发现一张紫色的便利贴。

John。不管发生什么，记住John。

他起来，穿上他蓝色的旧晨袍，不系带子，任它在他下楼走进小后院时在他身体周围窸窣作响。

John在那里，坐在一张椅子上读着报。他手里没有捧着茶，Sherlock忘了泡茶。他也没给自己泡咖啡。

他走近到他身后，弯下腰去，嘴唇轻轻擦过John的太阳穴。

“我忘了告诉你我爱你。”

他没法看见，可他能感觉到John在微笑。

Sherlock醒来，在床头柜上发现了一张黄色的便利贴。

别费事泡茶了。

他起来，穿上他蓝色的旧晨袍，不系带子，任它在他下楼时在他身体周围窸窣作响。

房子里寂静无声。

-The End-
